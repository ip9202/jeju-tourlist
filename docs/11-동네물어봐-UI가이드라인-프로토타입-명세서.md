## 🎨 디자인 시스템 기본 원칙

### 설계 철학

"동네물어봐"는 제주도의 따뜻하고 친근한 커뮤니티 문화를 반영한 메신저 스타일 Q&A 플랫폼입니다.

#### 핵심 디자인 가치
1. **친근함 (Approachable)**: 누구나 쉽게 질문하고 답변할 수 있는 편안한 분위기
2. **실시간성 (Real-time)**: 즉시 소통 가능한 채팅 인터페이스
3. **신뢰성 (Trustworthy)**: 현지인 인증과 검증된 정보의 시각적 표현
4. **지역성 (Local)**: 제주도의 자연과 문화를 반영한 따뜻한 색감

#### UI/UX 패러다임
- **메신저 퍼스트**: 카카오톡, 인스타그램 DM 스타일의 자연스러운 대화형 인터페이스
- **모바일 우선**: 여행 중 실시간 사용을 고려한 모바일 최적화
- **단순함**: 복잡한 폼보다는 직관적인 메시지 입력 방식
- **즉시성**: 빠른 응답을 위한 원터치 액션과 알림 시스템

## 🏷️ 브랜드 아이덴티티

### 브랜드 컨셉
- **서비스명**: "동네물어봐" (AskLocal/NeighborAsk)
- **컨셉**: 제주도 여행자와 현지 주민을 연결하는 실시간 Q&A 커뮤니티
- **톤앤매너**: 따뜻하고 친근한, 신뢰할 수 있는, 제주도의 자연스러움

### 브랜드 키워드
- **1차**: 실시간, 친근함, 신뢰성, 지역성
- **2차**: 도움, 공유, 연결, 발견
- **3차**: 여행, 경험, 추천, 커뮤니티

### 로고 및 아이콘 시스템
```
💬 메시지 버블: 소통과 대화를 상징
🏠 집 아이콘: 현지인과 지역성을 표현
📍 위치 핀: 지역 기반 서비스 특성
⭐ 별 아이콘: 추천과 품질을 나타냄
🎯 타겟 아이콘: 정확한 정보 제공
```
